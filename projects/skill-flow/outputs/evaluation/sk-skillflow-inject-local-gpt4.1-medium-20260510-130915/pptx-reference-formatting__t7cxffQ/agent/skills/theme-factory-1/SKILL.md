# Theme Factory Skill

Provides professional font and color themes for artifacts. Each theme includes color palettes, font pairings, and visual identity.

## Workflow

1. Show `theme-showcase.pdf` to user
2. Ask for theme selection
3. Wait for confirmation
4. Read theme file from `themes/` directory
5. Apply colors and fonts to artifact

## Available Themes

1. **Ocean Depths** - Professional maritime theme
2. **Sunset Boulevard** - Warm sunset colors
3. **Forest Canopy** - Natural earth tones
4. **Modern Minimalist** - Contemporary grayscale
5. **Golden Hour** - Warm autumnal palette
6. **Arctic Frost** - Cool winter-inspired theme
7. **Desert Rose** - Sophisticated dusty tones
8. **Tech Innovation** - Bold tech aesthetic
9. **Botanical Garden** - Fresh garden colors
10. **Midnight Galaxy** - Dramatic cosmic tones

## Custom Themes

If existing themes don't fit, generate a new theme with:

- Descriptive name reflecting color/font combinations
- Colors and fonts based on user requirements
- Show for review before applying
