# Data Formats & Conventions

### Data Formats
- **Input/Output**: JSON Lines (`.jsonl`).
- **Resuming Key**: Most scripts track progress using an `idx` field in the JSON objects.
- **Reasoning**: Reasoning traces are enclosed in `<think>...</think>` tags.
