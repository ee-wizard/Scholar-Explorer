# 図解・グラフガイドライン

**責務**: 14種の図解タイプ・グラフ選択・データ可視化・**メッセージの伝え方**

---

## 1. 図解の基本原則

### 重要: 図は「主張」を伝える手段

**図だけでは伝わらない。図から何を言いたいかを必ず明示する。**

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  【スライド構成】                                        │
│                                                         │
│  ┌─────────────────────────────────────────────┐       │
│  │ タイトル: 市場シェアの推移（主張なし・簡潔）    │       │
│  └─────────────────────────────────────────────┘       │
│                                                         │
│  ┌─────────────────────────────────────────────┐       │
│  │ メッセージ: しかし、競合参入により当社シェアが   │       │
│  │            20%低下している（50文字以内・主張）  │       │
│  └─────────────────────────────────────────────┘       │
│                                                         │
│  ┌─────────────────────────────────────────────┐       │
│  │                                             │       │
│  │            【図解・グラフ】                  │       │
│  │         （メッセージを視覚的に補強）         │       │
│  │                                             │       │
│  └─────────────────────────────────────────────┘       │
│                                                         │
│  ┌─────────────────────────────────────────────┐       │
│  │ 注釈: 重要ポイントを矢印・色・ラベルで強調    │       │
│  └─────────────────────────────────────────────┘       │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 図解スライドの3層構造

| 層 | 役割 | ルール |
|----|------|--------|
| **タイトル** | 図の内容を示す | 主張なし、20文字以内 |
| **メッセージ** | 図から言いたいことを主張 | 50文字以内、接続詞で流れを作る |
| **図解本体** | メッセージを視覚的に補強 | 注釈・強調で要点を明示 |

### 図解への注釈・強調の入れ方

| 強調方法 | 用途 | 例 |
|---------|------|-----|
| **色での強調** | 重要データを目立たせる | 当社シェアのみ`--sakura-pink` |
| **矢印での指示** | 注目すべき箇所を示す | 「ここが問題」→ |
| **ラベル追加** | 数値や意味を明示 | 「-20%」「目標達成」 |
| **囲み・ハイライト** | 範囲を強調 | 破線で囲む、背景色 |
| **吹き出し** | 補足説明を追加 | 「競合A社参入」 |

### 図解の目的

| 目的 | 効果 | 選ぶべき図解 |
|------|------|-------------|
| 構造を示す | 全体像の理解 | ロジックツリー、ピラミッド |
| 流れを示す | 順序の理解 | フロー型、回転型 |
| 比較する | 違いの理解 | 対比型、マトリックス |
| 分類する | 整理の理解 | 列挙型、表型 |
| 位置づける | 関係の理解 | 4象限型 |

## 2. 14種の図解タイプ

### 2.1 ロジックツリー型

**用途**: 論理構造・階層関係・分解

```
┌─────────────────────────────────────────────┐
│              メインテーマ                    │
│                   │                         │
│     ┌─────────────┼─────────────┐           │
│     ↓             ↓             ↓           │
│  ┌─────┐     ┌─────┐     ┌─────┐           │
│  │ 要素1│     │ 要素2│     │ 要素3│           │
│  └─────┘     └─────┘     └─────┘           │
│     │           │           │               │
│   ┌─┴─┐       ┌─┴─┐       ┌─┴─┐           │
│   ↓   ↓       ↓   ↓       ↓   ↓           │
│  1-1 1-2     2-1 2-2     3-1 3-2           │
└─────────────────────────────────────────────┘
```

**適用場面**:
- 問題の原因分析
- 組織構造の説明
- 意思決定の分岐

### 2.2 列挙型

**用途**: 複数要素の並列表示

```
┌─────────────────────────────────────────────┐
│                                             │
│  ┌───────┐  ┌───────┐  ┌───────┐           │
│  │       │  │       │  │       │           │
│  │ 要素1  │  │ 要素2  │  │ 要素3  │           │
│  │       │  │       │  │       │           │
│  └───────┘  └───────┘  └───────┘           │
│                                             │
│    説明1       説明2       説明3            │
│                                             │
└─────────────────────────────────────────────┘
```

**適用場面**:
- 特徴・メリットの列挙
- 選択肢の提示
- 並列する概念の説明

### 2.3 背景型

**用途**: 背景から結論への流れ

```
┌─────────────────────────────────────────────┐
│                                             │
│  ┌─────────────────────────────────┐       │
│  │          背景・前提              │       │
│  └─────────────────────────────────┘       │
│                   │                         │
│                   ▼                         │
│  ┌─────────────────────────────────┐       │
│  │          結論・主張              │       │
│  └─────────────────────────────────┘       │
│                                             │
└─────────────────────────────────────────────┘
```

**適用場面**:
- 状況説明から提案へ
- 前提条件の明示
- 論理的な導入

### 2.4 拡散型

**用途**: 一つの要素から複数への展開

```
┌─────────────────────────────────────────────┐
│                                             │
│              ┌───────┐                      │
│              │ 中心  │                      │
│              └───┬───┘                      │
│         ┌───────┼───────┐                  │
│         ↓       ↓       ↓                  │
│     ┌─────┐ ┌─────┐ ┌─────┐               │
│     │ A   │ │ B   │ │ C   │               │
│     └─────┘ └─────┘ └─────┘               │
│         ↓       ↓       ↓                  │
│       詳細1   詳細2   詳細3                 │
│                                             │
└─────────────────────────────────────────────┘
```

**適用場面**:
- 影響範囲の説明
- 波及効果の表現
- 一対多の関係

### 2.5 合流型

**用途**: 複数要素から一つへの収束

```
┌─────────────────────────────────────────────┐
│                                             │
│     ┌─────┐ ┌─────┐ ┌─────┐               │
│     │ A   │ │ B   │ │ C   │               │
│     └──┬──┘ └──┬──┘ └──┬──┘               │
│        │       │       │                   │
│        └───────┼───────┘                   │
│                ↓                            │
│           ┌────────┐                        │
│           │ 結論   │                        │
│           └────────┘                        │
│                                             │
└─────────────────────────────────────────────┘
```

**適用場面**:
- 要因の統合
- シナジー効果
- 多対一の関係

### 2.6 フロー型

**用途**: 直線的なプロセス・手順

```
┌─────────────────────────────────────────────┐
│                                             │
│  ┌────┐    ┌────┐    ┌────┐    ┌────┐     │
│  │ 1  │ ─→ │ 2  │ ─→ │ 3  │ ─→ │ 4  │     │
│  └────┘    └────┘    └────┘    └────┘     │
│                                             │
│   開始     処理1     処理2     完了        │
│                                             │
└─────────────────────────────────────────────┘
```

**適用場面**:
- 業務フローの説明
- 手順の案内
- タイムラインの表示

### 2.7 回転型（サイクル）

**用途**: 循環するプロセス

```
┌─────────────────────────────────────────────┐
│                                             │
│              ┌─────┐                        │
│              │  1  │                        │
│              └──┬──┘                        │
│           ↗        ↘                       │
│      ┌─────┐        ┌─────┐                │
│      │  4  │        │  2  │                │
│      └─────┘        └─────┘                │
│           ↖        ↙                       │
│              ┌─────┐                        │
│              │  3  │                        │
│              └─────┘                        │
│                                             │
└─────────────────────────────────────────────┘
```

**適用場面**:
- PDCAサイクル
- 継続的改善プロセス
- 季節・周期的な活動

### 2.8 上昇型

**用途**: 成長・向上・段階的発展

```
┌─────────────────────────────────────────────┐
│                                             │
│                          ┌─────┐           │
│                          │ 目標 │           │
│                     ┌────┴─────┘           │
│                ┌────┴────┐                  │
│           ┌────┴────┐    │ STEP3           │
│      ┌────┴────┐    │    │                 │
│ ┌────┴────┐    │ STEP2  │                  │
│ │ 現状    │ STEP1│    │                    │
│ └─────────┴─────┴─────┴─────┘              │
│                                             │
└─────────────────────────────────────────────┘
```

**適用場面**:
- 成長ロードマップ
- スキルアップの段階
- 売上目標の推移

### 2.9 対比型

**用途**: 2つの要素の比較

```
┌─────────────────────────────────────────────┐
│                                             │
│  ┌────────────┐    ┌────────────┐          │
│  │            │    │            │          │
│  │   Before   │ VS │   After    │          │
│  │            │    │            │          │
│  ├────────────┤    ├────────────┤          │
│  │ ・特徴1    │    │ ・特徴1    │          │
│  │ ・特徴2    │    │ ・特徴2    │          │
│  │ ・特徴3    │    │ ・特徴3    │          │
│  └────────────┘    └────────────┘          │
│                                             │
└─────────────────────────────────────────────┘
```

**適用場面**:
- Before/After
- 競合比較
- 新旧比較

### 2.10 マトリックス型

**用途**: 2軸による分類・整理

```
┌─────────────────────────────────────────────┐
│                                             │
│          │   軸A-高    │   軸A-低    │      │
│  ────────┼────────────┼────────────┤      │
│  軸B-高  │    象限1    │    象限2    │      │
│  ────────┼────────────┼────────────┤      │
│  軸B-低  │    象限3    │    象限4    │      │
│                                             │
└─────────────────────────────────────────────┘
```

**適用場面**:
- 優先度の整理（緊急度×重要度）
- ポジショニングマップ
- リスク評価

### 2.11 表型

**用途**: 詳細情報の整理・比較

```
┌─────────────────────────────────────────────┐
│                                             │
```

**用途**: 2軸による位置づけ・マッピング

## 3. 図解タイプ選択ガイド

### 目的別選択表

| 伝えたいこと | 第1候補 | 第2候補 |
|-------------|--------|--------|
| 論理構造 | ロジックツリー | ピラミッド |
| 並列要素 | 列挙型 | 表型 |
| 順序・流れ | フロー型 | 上昇型 |
| 循環 | 回転型 | フロー型（環状） |
| 比較 | 対比型 | マトリックス |
| 分類 | マトリックス | 4象限型 |
| 計画 | ガントチャート | フロー型 |
| 因果関係 | 背景型 | 合流型 |
| 影響範囲 | 拡散型 | ロジックツリー |
| 位置づけ | 4象限型 | マトリックス |

## 4. グラフの基本原則

### グラフの目的

| 比較タイプ | 答える質問 | 推奨グラフ |
|-----------|-----------|-----------|
| 内訳比較 | 構成要素は何か？ | 積み上げグラフ、円グラフ |
| 量の比較 | どちらが大きいか？ | 横棒グラフ |
| 変化の比較 | どう推移したか？ | 縦棒グラフ、折れ線グラフ |
| 傾向の比較 | トレンドは？ | 折れ線グラフ |
| 相関の比較 | 関係はあるか？ | 散布図 |

## 6. グラフ選択フローチャート

```
何を伝えたいか？
    │
    ├─→ 内訳・構成 ──→ 積み上げグラフ / 円グラフ
    │
    ├─→ 量の比較 ──→ カテゴリが多い？
    │                   ├─→ Yes ──→ 横棒グラフ
    │                   └─→ No ──→ 縦棒グラフ
    │
    ├─→ 時系列変化 ──→ 連続的？
    │                   ├─→ Yes ──→ 折れ線グラフ
    │                   └─→ No ──→ 縦棒グラフ
    │
    └─→ 相関関係 ──→ 散布図
```

| 強調したい要素 | `--wave-blue` または `--sakura-pink` |
| それ以外 | グレー系（`--fg-dim`） |
| ポジティブ | `--wave-aqua`（青緑系） |
| ネガティブ | `--sakura-pink`（ピンク系） |
