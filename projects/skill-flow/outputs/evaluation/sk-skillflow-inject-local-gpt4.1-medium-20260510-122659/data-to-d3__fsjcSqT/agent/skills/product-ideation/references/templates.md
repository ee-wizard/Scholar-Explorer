## スライド化時の注意

baoyu-slide-deck でスライド化する際の推奨設定

```bash
/baoyu-slide-deck docs/product-concept.md --audience executives --style corporate
```

スライドで必ず含めるべき内容は3つ。一言キャッチコピー（タイトルスライド）、ビッグテックとの差別化（Why not Google?）、参入障壁（なぜ勝てるか）。

VCが必ず聞く質問への回答を準備しておく。「なぜGoogle/Microsoftがやらないの？」「日本特化で本当に大丈夫？」「参入障壁は何？」
