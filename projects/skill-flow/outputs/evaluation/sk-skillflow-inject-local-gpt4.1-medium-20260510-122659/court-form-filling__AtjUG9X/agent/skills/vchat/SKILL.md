# VChat - 公众号一键发布

## 💡 简介

这是一个快捷命令，输入 `/vchat` 即可启动公众号文章发布流程。

## 🚀 使用方式

### 方式1：完整流程（从选题到发布）

```
/vchat 数字货币
```

或

```
/vchat AI工具
```

**自动执行**：
1. 扫描相关热点
2. 推荐优质选题
3. 撰写文章
4. AI去痕
5. 质量评分
6. 生成封面
7. 发布草稿

**自动执行**：
1. AI去痕

3. 生成封面

## 📋 命令速查

|------|------|
| `/vchat [主题]` | 完整流程（选题→发布） |

## 🎯 执行逻辑

当用户输入 `/vchat [参数]` 时：

### 1. 解析参数

- 无参数或只有主题 → 完整流程
- "发布" + 文章路径 → 直接发布
- "选题" + 主题 → 选题分析
- "热点" → 扫描热点

### 2. 调用对应技能

- `news-aggregator-skill` - 热点扫描
- `topic-picker` - 选题评估
- `humanizer-zh` - AI去痕
- `article-scorer` - 质量评分
- `wechat-draft (NotebookLM)` - 🆕 **AI封面生成 + 发布草稿**

### 3. 统一输出

- 文章保存到：`D:\Obsidian_AI_WeChat_Kit\公众号发布区\`
- 封面保存到：同目录
- 返回草稿链接

## 🎨 特色功能

### 1. 🆕 NotebookLM AI 封面生成

**新特性**：使用 NotebookLM AI 自动生成专业信息图

**优势**：
- ✅ AI 分析文章内容，提取关键信息
- ✅ 专业设计风格，配色协调
- ✅ 自动适配文章主题（对比/数据/趋势）
- ✅ 比 Canvas 生成的更美观、更专业
- ✅ 零人工设计，3-5分钟自动完成

**工作流程**：
```
文章内容 → NotebookLM → AI分析 → 生成信息图 → 自动下载 → 上传公众号
```

**备用方案**：
- 如果 NotebookLM 速率限制，自动跳过封面生成
- 如果认证过期，提示重新登录
- 不会阻塞发布流程

### 2. 质量把关

- 评分<80分自动暂停
- 给出具体改进建议
- 二次评分确认

### 3. 自动保存

- 所有文章统一保存到发布区
- 文件名自动规范化
- 封面与文章同名

## ⚠️ 注意事项

1. **首次使用**：
   - 配置环境变量（AppID/AppSecret）
   - 🆕 NotebookLM 认证：`notebooklm login`
2. **IP白名单**：需要在公众号后台添加当前IP
3. **封面生成**：
   - 首次使用需要 3-5 分钟生成封面
   - 如果失败，会自动跳过封面生成
4. **发布前预览**：草稿创建后务必预览再发布
5. **备份原文**：系统会自动保存，无需手动备份
