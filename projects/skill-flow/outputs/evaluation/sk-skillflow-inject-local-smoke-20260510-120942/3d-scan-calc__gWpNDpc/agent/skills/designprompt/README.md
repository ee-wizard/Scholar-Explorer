# designprompt

AI驱动的智能设计系统构建器，基于30+专业设计风格自动为项目推荐和应用最合适的设计系统。

## 特性

- **智能推荐**：AI自动分析项目特征，推荐最合适的2-3个设计风格
- **30+专业风格**：涵盖从极简主义到赛博朋克的各类设计风格
- **完整设计规范**：每个风格包含颜色、字体、组件、动效等完整规范
- **统一图标系统**：所有风格使用 Lucide 图标库，1000+ 高质量图标
- **灵活选择**：支持AI推荐，也支持用户直接指定风格
- **可扩展**：轻松添加新的设计风格

## 使用方法

### 方式一：AI智能推荐（推荐）

直接描述你的项目，AI会自动推荐合适的风格：

```
我要做一个高端珠宝品牌的官网
```

AI会分析项目特征并推荐：
- Luxury（奢华风格）
- Monochrome（极简黑白）
- Art Deco（装饰艺术）

### 方式二：直接指定风格

如果你已经知道想要的风格：

```
用 Bauhaus 风格做一个设计工作室的网站
```

```
我想用 Cyberpunk 风格创建一个游戏平台
```

## 工作原理

### 1. 项目分析

AI分析你的项目：
- 行业类型（电商、金融、娱乐...）
- 目标受众（高端用户、年轻人、专业人士...）
- 品牌调性（专业、活泼、奢华、极简...）
- 功能需求（营销页、管理后台、内容平台...）

### 2. 智能匹配

基于30+风格的元数据进行智能匹配：
- 行业适配度
- 情感调性契合度
- 功能需求匹配度

### 3. 风格推荐

推荐2-3个最合适的风格，包含：
- 推荐理由
- 核心特征
- 适用场景

### 4. 系统应用

加载完整设计系统并实现界面：
- 设计Token（颜色、字体、间距...）
- 组件规范（按钮、卡片、表单...）
- 布局系统
- 动效规范
- 响应式设计
- 可访问性
- 图标系统（Lucide 图标库）

## 图标系统

所有30个设计风格统一使用 **Lucide** 图标库：

### 为什么选择 Lucide

- ✅ 现代简洁，适合所有设计风格
- ✅ 1000+ 高质量图标，持续更新
- ✅ 完美的 tree-shaking 支持
- ✅ 跨框架：React、Vue、Svelte
- ✅ MIT 开源协议

### 快速使用

**React**：
```bash
npm install lucide-react
```

```tsx
import { ChevronDown, Check, Star } from 'lucide-react'

<ChevronDown size={20} strokeWidth={2} />
```

**Vue**：
```bash
npm install lucide-vue-next
```

详细配置和不同风格的图标处理方式请参考 `SKILL.md` 中的"图标库规范"章节。

## 优势

相比传统的 `frontend-design` 技能：

| 特性 | frontend-design | designprompt |
|------|----------------|----------------------|
| 风格选择 | 需要人工创造 | 30+专业风格库 |
| 一致性 | 依赖AI理解 | 完整设计系统规范 |
| 可预测性 | 每次可能不同 | 遵循标准设计系统 |
| 学习成本 | 高 | 低（AI自动推荐） |
| 扩展性 | 有限 | 可持续添加新风格 |

## 扩展技能

参见 `EXTEND.md` 了解如何：
- 添加新的设计风格
- 自定义风格元数据
- 贡献设计系统

```
- 等宽字体

### 2. Modern Dark（推荐指数：⭐⭐⭐⭐）
**推荐理由**：现代暗黑主题，专业且高效

你更倾向于哪个风格？或者我可以展示其他选项。

AI: 好的！我将使用 Terminal 设计系统。这个风格将为你的开发者工具创造一个充满极客感的界面... [开始实现]
```
