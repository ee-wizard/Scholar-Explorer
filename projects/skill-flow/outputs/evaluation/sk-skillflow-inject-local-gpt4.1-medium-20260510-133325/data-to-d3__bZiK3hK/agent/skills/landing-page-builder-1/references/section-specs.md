# Section Specifications

Each section has a job. If it doesn't earn its place, cut it.

## 1. Hero
**Job:** Hook in 3 seconds. Promise a transformation.

| Element | Spec |
|---------|------|
| Headline | Outcome-first. 6-10 words max. |
| Subhead | Who + What. One sentence. |
| CTA | Single primary action. Verb + benefit. |
| Visual | Product shot, demo, or ambient background |
| Proof hint | Logo bar, stat badge, or "Trusted by X" |

**Effects:** Background (Aurora, Particles, Vanta), Text Generate/Reveal, Spotlight

---

## 2. Problem
**Job:** Validate pain. Create tension.

| Element | Spec |
|---------|------|
| Header | "The [X] problem" or question format |
| Pain points | 3-4 cards using customer language |
| Tone | Empathetic, not fear-mongering |

**Effects:** Scroll Reveal, Card hover (3D, Wobble), Tracing Beam

---

## 3. Solution / Demo
**Job:** Release tension. Show the product.

| Element | Spec |
|---------|------|
| Header | "Introducing [Product]" or outcome statement |
| Demo | Interactive embed, video, or scrollytelling walkthrough |
| Caption | One sentence explaining what they're seeing |

**Effects:** Macbook Scroll, Container Scroll Animation, Compare, Device Mockups

---

## 4. Benefits
**Job:** Translate features into outcomes.

| Element | Spec |
|---------|------|
| Layout | 6 cards (Bento or grid) |
| Card format | Icon + Outcome headline + 1-line proof |
| Hierarchy | Lead with top 2 most compelling |

**Effects:** Bento Grid, Magic Card, 3D Card Effect, Staggered reveal

---

## 5. How It Works
**Job:** Reduce complexity. Show simplicity.

| Element | Spec |
|---------|------|
| Steps | 3-4 max. Numbered or visual flow. |
| Format | Icon + Short title + One sentence |

**Effects:** Timeline, Tracing Beam, Sticky Scroll Reveal, Number animations

---

## 6. Social Proof
**Job:** Transfer trust.

| Element | Spec |
|---------|------|
| Logos | 6-12 recognizable names. Grayscale unless brand colors fit. |
| Testimonials | 3 quotes with name, title, photo. Outcome-focused. |
| Stats | 2-3 metrics with context |

**Effects:** Marquee, Infinite Moving Cards, Card Stack, Avatar Circles, Number Ticker

---

## 7. Comparison (Optional)
**Job:** Frame the decision.

| Element | Spec |
|---------|------|
| Format | 2-column: Old way vs Your way |
| Rows | 4-6 comparison points |
| Tone | Factual, not disparaging |

**Effects:** Compare component, Focus Cards

---

## 8. FAQ
**Job:** Handle objections. Reduce friction.

| Element | Spec |
|---------|------|
| Questions | 5-6 real objections |
| Format | Accordion. Concise answers. |

**Effects:** Expandable Card, Framer Motion layout

---

## 9. Final CTA
**Job:** Close. No distractions.

| Element | Spec |
|---------|------|
| Headline | Action-oriented. Repeat core promise. |
| CTA | Same as hero. Optional secondary link. |
| Reassurance | "No credit card required" / "Free trial" |

**Effects:** Background Beams, Spotlight, Vortex, or clean gradient
