## 2. Scroll Effects & Scrollytelling

Effects that create narrative experiences through scroll-based animations.

### Core Scroll Libraries

| Library | Description | License | Best For |
|---------|-------------|---------|----------|
| **[GSAP ScrollTrigger](https://gsap.com/scroll/)** | Industry-standard scroll animation library with pinning, scrubbing, and timeline control | Free (standard) | Complex scrollytelling, professional sites |
| **[Lenis](https://github.com/darkroomengineering/lenis)** | Lightweight smooth scroll library, perfect foundation for scroll animations | MIT | Smooth scrolling base layer |
| **[@bsmnt/scrollytelling](https://github.com/basementstudio/scrollytelling)** | React wrapper for GSAP ScrollTrigger with position-based animations | MIT | React scrollytelling |
| **[Framer Motion scroll()](https://motion.dev/docs/react-scroll-animations)** | React scroll animations with useScroll and useTransform hooks | MIT | React projects |
